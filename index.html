Key command class toggler

Add/remove a class on the html tag by customizable key command

Demo: Press Control + 8 to toggle class

Initialize (example)

Options (default)

  • toggleClass: "example-class" // String
  • keyOne: "control" // Key String
  • keyTwo: "7" // Key String
  • activeMessage: "Class is activated" // String
  • deactiveMessage: "Class is deactivated" // String
